# AGENT.md
ユーザーとはすべて日本語でやり取りしてください。

## このファイルの目的
- このリポジトリで作業する自動化エージェントや新しい貢献者のための行動指針を明文化しています。
- 一貫したビルド・デプロイ手順と、チームが期待するコード品質を満たすためのチェックリストを提供します。


## プロダクト概要
- **ランタイム**: Node.js 22（Cloud Functions Gen 2）。
- **言語とビルド**: TypeScript（`tsc`）で `src/`→`lib/` にコンパイル。`strict` モードと ES2022 をターゲット。
- **開発ツール**: Functions Framework（ローカル）、Biome（lint/format）、`gcloud` CLI（デプロイ）。

## コーディング基準
- `tsconfig.json` の `strict` オプションを守り、エラーを抑制せず修正。
- Biome のフォーマット設定（スペース 2、ダブルクオート）と lint ルールに従う。
- ログは量を抑えて Cloud Functions のモニタリングに負荷をかけない。
